# Operational Usefulness of Alerts from Anomaly-Based Intrusion Detection Systems

This repository contains the code and setup for a university project focused on evaluating the operational usefulness of alerts generated by an anomaly-based intrusion detection system (IDS) for detecting port scanning attacks.

## Project Overview

In this project, we:

1. Developed an anomaly-based approach to detect port scans
2. Generated alerts in EVE JSON format
3. Integrated DFIR-IRIS to create cases and enrich alerts with additional information

The project focuses not only on detecting suspicious network activity, but also on assessing how useful the generated alerts are for operational security workflows and incident handling.

## Requirements

To run the project, ensure you have the following installed:

- Docker
- Python 3.10.1
- Python libraries listed in `requirements.txt`

## Installation

Clone the repository:

```bash
git clone https://github.com/t-kolev/Anomaly-Based-IDS-Alert-Evaluation

cd Anomaly-Based-IDS-Alert-Evaluation
```

Install the required Python libraries:

```bash
pip install -r requirements.txt
```

## Folder Structure

- `IDS/` – contains the anomaly-based intrusion detection logic for detecting port scans and generating alerts
- `CaseManagement/` – contains the DFIR-IRIS integration, including scripts and templates for case creation and alert enrichment
- `Report/` – contains the final project report with methodology, implementation details, and results

## Intended Workflow

1. Install and run DFIR-IRIS by following the guide in `CaseManagement/IRIS_README.md`
2. Replace the API keys in `CaseManagement/setup.py` and `CaseManagement/create_case.py` with your own
3. Add the PCAP file or files you want to analyze to `IDS/dataset/`
4. Follow the IDS instructions
5. Follow the IRIS instructions

## Report

The repository includes a final report describing:

- the anomaly-based intrusion detection methodology
- implementation challenges
- results and insights from analyzing IDS alerts
- the alert enrichment process
- the use of DFIR-IRIS for more manageable alert processing
